Why the Zero System project?

I Christoph Dorn have been continuously searching for the ultimate development environment and system. Zero System is my attempt to bring declarative based system composition to NodeJS and JavaScript application development. I believe we must start declaring how we wish to compose our applications (which are really systems) rather than code them. Declarative composition seems to be the underpinning of component-based development and demands more attention.
